Tutvustus

Märgend - ja stiililehe keelte põhiteadmiste õpetamine ja kontrollimine veebikeskkonnas, mis on mõeldud Noorem Tarkvaraarendaja 1 kursuse ja IT süsteemi spetsialisti programmeerimise suunamooduli õpilastele.
Vajuta HTML või CSS ikooni peale, et saada lühitutvustust, mis need on.

HTML

HTML on keel, milles märgendatakse veebilehti. HTML-dokumentide (failide) veebist allalaadimiseks ja vaatamiseks kasutatakse veebibrauserit. Selline elektroonilise dokumendi märgendamise mõte on lahutada sisu, vaade jms objekti omadused üksteisest. HTML paneb paika üksnes dokumendi struktuuri, kuid võimaldab veebiehele kaasata skripte, eeskätt JavaScripti ja CSS-i, millega kirjeldatakse lehe kujundus.

CSS

CSS (Cascadin Style Sheet) e. Kaskaadsed stiililehed. S.o. meetod mida kasutatakse veebilehtede toimetamisel. Stiilid määravad kindlaks mismoodi kuvatakse Html elemendid. Kogu veebisaidi kujundus on viidud ühte dokumenti ning selles dokumendis tehtud muudatused kajastuvad korraga üle terve saidi. Kasutades stiile, hoiad kokku oma aega.

HTML:

Kuidas kirjutada HTML koodi

Html koodi kirjutamiseks tuleb installeerida arvutisse kas visual studio code, brackets, sublime või mingi muu programm ja salvestada fail .html laiendiga. Näiteks salvesta fail index.html-ina

Ülesanne: Installeerida üks koodi kirjutamis programm (sublime, brackets) ja salvestada fail html-ina.

Lisa materjal: https://www.w3schools.com/html/html_intro.asp

Kuidas alustada HTML koodi kirjutamist

HTML dokumenti peab alati alustama <!DOCTYPE html>. edasi tuleb <html> ja </html> ning <body> </body>. h1-ga on millega kirjutatakse pealkirju, p-ga kirjutatakse paragrahve. Koodid algavad normaal kujuga nagu näiteks <html> ja lõppeb </html-iga>.
Näide <!DOCTYPE html> <html> <body> <h1> pealkiri </body> </html>

Ülesanne: Kirjutada loodud html faili html-i vajalikud koodid, panna pealkirjaks oma kursuse nimi ning selle alla paragrahv oma nimega.

Lisa materjal: https://www.w3schools.com/html/html_basic.asp

Kuidas kuvada enda HTML koodi

Koodi on võimalik kuvada kui tõstad oma html faili enda brauserisse (google chrome, internet explorer jne). Samuti on võimalik ka kuvada internetis olevate veebilehekülgede koodi. Seda saab teha kui parem klickida lehele ja vajutada source code.

Ülesanne: tõsta oma loodud fail brauserisse ja vaata, kas töötab.

lisa materjal: https://www.w3schools.com/html/html_editors.asp

Kuidas muuta tausta värvi ja teksti värvi

Tausta värvi muutmiseks tuleb lisada <body> sisse style="background-color:blue;" Lisades selle muudab tausta värvi siniseks. Sinise asemel on võimalik panna teisi värve. Teksti värvi muutmine on täpselt samamoodi, ainult et tuleb kirjutada color mitte background-color.

Ülesanne: Muuda oma tausta ja teksti värve.

Näide: <body style="background-color: blue;"> </body>

Lisa materjal: https://www.w3schools.com/html/html_styles.asp

Kuidas kommenteerida oma koodi

Koodi kommenteerimiseks tuleb kirjutada nii <!-- kirjuta kommentaar -->

Ülesanne: kommenteeri kõike, mida sa oled kirjutanud ja katsetanud

Näide: <!----- <p> proovimine </p> ------>

Lisa materjal: https://www.w3schools.com/html/html_comments.asp

Kokkuvõtlik test

Loo uus html fail, lisa html-i vajalikud koodid, sisesta oma nimi ja kursus ning muuda tausta ja teksti värv

CSS:

Kuidas sisestada CSS-i HTML-i koodi

Väline stiilileht Välise stiilis lehe abil saate muuta ühe veebilehe välimust, muutes vaid ühte faili! Iga lehekülg peab sisaldama viidet elementi <link> välisele stiililehe failile. <link> element liigub <head> sektsiooni:
<head> <link rel="stylesheet" type="text/css" href="mystyle.css"> </head>

Ülesanne: Tee uus väline stiilileht ja link-i HTML-iga

Lisa materjal: https://www.w3schools.com/css/css_howto.asp

Taustavärvi ja tekstivärvi lisamine CSS-is

Saad lisada tastavärvi HTML-i elementidele:

<h1 style="background-color:Blue;"> </h1>

Saad lisada tekstivärvi HTML-i elementidele:

<h1 style="color:Blue;"> </h1>

Ülesanne: Võta lahti oma notepad ja proovi tulemusi

Lisa materjal: https://www.w3schools.com/css/css_colors.asp

Taustavärvi ja pildi lisamine tervele lehele

Kas lisada HTML-i <head> </head> vahele või lisad eradi stiililehele:

Taustavärvi lisamine: h1 {background-color: green;}

Pildi lisamine: body {background-image: url("gradient_bg.png");}

Ülesanne: Proovi antud asju kas eraldi stiililehe peal või HTML-i <head> </head> vahele ja pildi lisamisel proovi saada ta keskele

Lisa materjal: https://www.w3schools.com/css/css_background.asp

Elemendi laiuse ja kõrguse muutmine kas protsentides või pikslites

div {height: 200px;width: 50%;background-color: powderblue;}

Ülesanne: Proovi muuta elemendi kõrgust ja laiust. Tee pilti väiksemaks või kõrgemaks

Lisa materjal: https://www.w3schools.com/css/css_background.asp

CSS-is padding ja margin-i lisamine

Padding on see ala, mis on elemendi piirjoone ja elemendi sisu vahel.

Margin on ala väljaspool igat elementi, mis ei lase teistel elementidel enda ligi.

div{background-color: grey;padding: 25px;margin: 25px;}

Ülesanne: Vaata mis saab. Lisa tausavärv, muuda tema paddingut ja marginit

Lisa materjal: https://www.w3schools.com/css/css_boxmodel.asp

Kokkuvõtlik test

Loo uus stiilileht ja lisa taustavärv tervelehele, tekstile. Muuda ka sisu paddingut ja marginit
